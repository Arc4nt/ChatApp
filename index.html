What is it?

Chatapp is a single page application of a real time chat. ChatApp contains the basic features of existing chats such as add and delete contacts, create groups, add and remove users to the groups, know when a contact is connected and also allows to send scribbles.

Technologies used:

This application was created using different technologies such as: Angularjs, Angular Materials, WebSockets, Ajax, HTML5 and CSS3 for the front-end. Whereas in the back-end technologies as: Nodejs, express, socketIO, mongodb, promises, and several middle-ware as cookie and body parser.

Why NodeJS and socketIO?

NodeJS has been coming up recently and can provide a pretty good response without the necessity of expensive hardware, and taking advantage of websockets and socketIO a live connection can be created to enable the server sending packages to the clients without the necessity that the client ask for it before. Also in the worst case scenario, if the client cant handle websockets it would fallback to polling.
